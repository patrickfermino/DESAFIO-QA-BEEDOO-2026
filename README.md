# DESAFIO-QA-BEEDOO-2026
Desafio Vaga QA
Objetivo da aplicação

A aplicação tem como objetivo permitir o cadastro e a visualização de cursos.
O usuário pode criar um novo curso preenchendo os campos do formulário e depois visualizar os cursos cadastrados em uma lista.

Principais fluxos disponíveis

Os principais fluxos identificados na aplicação são:

Cadastro de cursos
Permite que o usuário preencha as informações de um curso e realize o cadastro no sistema.

Listagem de cursos
Mostra os cursos que já foram cadastrados, permitindo visualizar as informações registradas.

Pontos críticos para teste

* Alguns pontos que considero mais críticos para teste são:

* Validação dos campos no cadastro de cursos

* Funcionamento correto do cadastro de novos cursos

* Exibição correta dos cursos na listagem

* Comportamento do sistema quando campos obrigatórios não são preenchidos

* Possíveis erros ao acessar cursos recém criados
